# Monte Carlo Risk Analysis

This page summarizes Finbot's Monte Carlo risk-analysis surface and points to
the interactive notebook that demonstrates the workflow.

## Overview

Finbot's Monte Carlo tooling generates probabilistic paths, percentile bands,
and final-value distributions for portfolio analysis and scenario exploration.

## What It Supports

- single-asset Monte Carlo sampling
- portfolio-level distribution analysis
- percentile and downside framing for risk communication
- exploratory retirement and allocation scenario work
