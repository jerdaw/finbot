# Configuration API

This page is a public wrapper for Finbot's configuration surface.

## Scope

- Dynaconf-backed settings loading
- environment selection via `DYNACONF_ENV`
- lazy accessors for API keys and threading settings
- failure-tolerant host and runtime metadata access
