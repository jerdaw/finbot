# Monte Carlo Visualization

Finbot includes lightweight plotting helpers for exploratory Monte Carlo output
inspection.

## API Reference

### plot_trials

::: finbot.services.simulation.monte_carlo.visualization.plot_trials
    options:
      show_root_heading: true
      show_source: true
      heading_level: 3

### plot_hist

::: finbot.services.simulation.monte_carlo.visualization.plot_hist
    options:
      show_root_heading: true
      show_source: true
      heading_level: 3
