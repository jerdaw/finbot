# run_backtest

The `run_backtest` helper provides a functional entry point for executing a
single backtest without constructing `BacktestRunner` manually.

## API Reference

::: finbot.services.backtesting.run_backtest.run_backtest
    options:
      show_root_heading: true
      show_source: true
      heading_level: 3
