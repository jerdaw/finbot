# Media Artifact Production Runbook

Runbook for Priority 7 manual media deliverables (videos and poster).

## Scope

1. P7.8 Finbot overview video.
2. P7.9 Project poster.
3. P7.20 Tutorial video set.
4. P7.25 Getting-started video.
5. P7.27 Contributing-guide video.

## Required Inputs

1. Objective and audience for each artifact.
2. Current product screenshots/demos.
3. Latest metrics from roadmap/docs for evidence claims.

## Production Workflow

### 1) Pre-Production

1. Choose artifact type (overview/tutorial/poster).
2. Copy template:
   - Video: `docs/templates/video-script-template.md`
   - Poster: `docs/templates/poster-outline.md`
3. Fill message goals, target length, and acceptance checklist.

### 2) Build

1. Capture visuals with consistent environment and data ranges.
2. Keep text claims aligned with tracked evidence in docs.
3. For videos, produce script + shot list before recording.

### 3) Quality Review

1. Verify dates, metrics, and feature claims against current docs.
2. Ensure disclaimers are visible where financial/health interpretation appears.
3. Validate accessibility:
   - readable font sizes
   - high contrast
   - captions or transcript for videos

### 4) Publish + Traceability

1. Publish to target platform.
2. Add publication URL and publish date to roadmap status rows.
3. Add source files under `assets/` or `docs/` where appropriate.

## Acceptance Checklist (All Artifacts)

1. Title and objective are explicit.
2. Claims are evidence-backed and date-stamped.
3. Includes limitations/disclaimer where needed.
4. Published URL recorded in planning docs.
5. Final artifact owner + revision date documented.
