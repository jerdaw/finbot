# Application Materials

This folder contains medical-school and portfolio-context writing derived from
the Finbot project.

These files are intentionally kept in the repository for transparency, but they
are **not** the same thing as the project's public product documentation.

## How To Read This Folder

- These documents explain how the project has been interpreted for
  application, interview, and portfolio purposes.
- They may use a more reflective tone than the main README or published docs.
- They should not be treated as the canonical source of public product counts
  or implementation status.

## Maintenance Rule

When a document in this folder uses public-facing metrics or counts, verify
them against [../METRICS.md](../METRICS.md) first. Prefer qualitative wording
when a metric is likely to drift.
